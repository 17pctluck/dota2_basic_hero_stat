17pctluck - Dota2 Heroes Stats (7.23e)

Visualizing and compare stats of Dota2's heroes such as Health, Attack Speed, Attack Damage. This can help you discover some interesting or useful stats. For example, Lifestealer has the highest attack speed. Or as a reference for quick peek at how much stats each hero has at any given level without having to fire up Dota2's client.

Each hero is represented by a card or rectangle. This card contains hero's name, attack capabilities (range or melee), range of attack, their full internal name and another attribute that show above which default to primary attribute of the hero.

Hover over a card will also give you most relevent stats (Name, STR, AGI, INT, HP, HP Regen, Mana, Mana regen, Attack damage, Armor, MS, value of current selected attribute).

To view other attribute you need to change the value of the Drop-down selector indicate by the word shown. The position of each hero on the grid are determined by their value of selected attribute and their primary attribute. For example, by selecting Agility as a shown attribute. At level 1, Juggernaut will appear first and follow by other heroes which have less agility from left to right.

When the position is needed to be fixed. This can be done by toggling 'Fix Grid Position' button. This will fix the position of each hero in place. There is also 'Reset' button which return everything to original state when the page is opened up.

The level slider can be use to change level of the heroes. By sliding the Level slider the hero's stats will be updated to specified level.

To check out some heroes. Tap tool is also come in handy with tap tool enable. Tapping or click on a card will select and highlight it. This might be useful seeing how selected heroes stats compare to other non-selected heroes. Multiple card can be selected by holding Shift key and exit highlighting by clicking on empty space in the plot or press Esc key.

How to use

  • Hover card for more information.
  • Use Drop-down selector to select desired attribute.
  • Use Level slider to change level of heroes.
  • Toggle 'Fix Grid Position' button to fix and unfix hero's position.
  • Click 'Reset' button to reset to default state.
  • Tap card with Tap tool active to highlight cards.

Tips: F11 will make your browser full screen which make interact with the plot easier. For low resolution display scrolling might be needed to see all content.

Known Issues

  • Stats might not load completely after the plot loaded, sliding level slider or change drop-down selector value will fix it.
  • On mobile, you might need to disable tap tool so the plot do not register both tap and hover at the same time. Disable tap tool by pressing tap tool icon on the right side of the plot.
